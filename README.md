# AI Bounty Board

A retro pixel-art western board where people post fun build prompts and vibe coders ship public demos in response. Townsfolk post bounties, Rangers submit demos, Sheriffs moderate. Winning Rangers earn a Sheriff Star.

This is the MVP. See [`docs/bounty-board-vibe-coders-prd_v2.md`](docs/bounty-board-vibe-coders-prd_v2.md) for the full product spec and [`AGENTS.md`](AGENTS.md) for stack and v0 scope.

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind v4 · Drizzle ORM · Supabase (Postgres + Auth + Storage) · deployed on Vercel.

## Develop

```bash
npm install
npm run dev
```

Then visit [http://localhost:3000](http://localhost:3000).
